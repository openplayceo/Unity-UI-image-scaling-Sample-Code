## Unity UI 이미지 크기 변경 샘플코드
<br/>

이 샘플은 게임의 Bar 타입 이미지의 길이를 런타임상에서 바꾸는 여러 방법들이 담겨져 있습니다.

HP 막대와 같이 게임 플레이 시 시시각각 변경이 요구될 때 사용되는 기본 기능들입니다.

<br/>

아래는 UI 이미지의 접근 및 변경이 되어야 하는 변수에 대한 나열입니다. 

샘플에는 총 4개의 변수 또는 속성에 대해 다룹니다.

+ Fill Amount
+ Scale
+ Width
+ Stretch
